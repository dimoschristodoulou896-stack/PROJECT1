# WordPress CLI Audit Tool

## Εισαγωγή

Το παρόν project αφορά την ανάπτυξη ενός εργαλείου γραμμής εντολών (CLI) για τον αυτοματοποιημένο έλεγχο ενός WordPress site σε τοπικό περιβάλλον (XAMPP).  
Στόχος είναι η αξιοποίηση τεχνολογιών ανοικτού λογισμικού για τον βασικό έλεγχο και τη συντήρηση ενός WordPress installation.

Το έργο υλοποιείται στο πλαίσιο εκπαιδευτικής εργασίας και βασίζεται στις αρχές της ανοικτότητας, της διαφάνειας και της σωστής τεκμηρίωσης.

## Επιχειρηματικό Μοντέλο (Managed Support)

Το εργαλείο μπορεί να αξιοποιηθεί για την παροχή υπηρεσιών Managed Support σε WordPress sites, όπως αυτοματοποιημένοι έλεγχοι και τεχνική υποστήριξη.  
Η εμπορική αξία προκύπτει από τις παρεχόμενες υπηρεσίες και όχι από τον ίδιο τον κώδικα.
